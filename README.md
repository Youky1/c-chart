# C-chart

> A library to draw simple chart in command line

Current support chart:

1. bar
2. scatter

## Bar

```typescript
import print from "c-chart";
print({
  type: "bar",
  data: [5, 7, 3, 2],
  direction: "y"
});
```

```
↑              _
|             | |
|             | |
|       _     | |
|      | |    | |
|      | |    | |
|      | |    | |     _
|      | |    | |    | |     _
|      | |    | |    | |    | |
+-------+------+------+------+---------→
```

Or you can change the direction by `direction: 'x'`

```typescript
import print from "c-chart";

  type: "bar",
  data: [5, 7, 3, 2]

```
